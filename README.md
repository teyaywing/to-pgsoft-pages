# to-pgsoft-pages

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内包含若干静态页面文件，主要用于存放和展示一些独立的网页内容，不针对特定域名或网站进行优化。

## 目录说明

```
to-pgsoft-pages/
├── README.md           # 本文件
├── index.html          # 入口页面（如有）
└── pages/              # 独立页面存放目录
    ├── page1.html
    ├── page2.html
    ...
```

- `index.html`：仓库根目录下的入口文件，用于跳转或展示页面列表。
- `pages/`：存放所有独立 HTML 页面的子目录。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开。

## 页面归档说明

本仓库中的页面均为历史归档或独立展示用途。它们可能包含以下特点：

- 不依赖外部资源（如 CDN、外部样式表或脚本），部分页面可能内联了样式和脚本。
- 页面之间无强关联，每个页面均可独立运行。
- 部分页面可能包含简单的交互或布局示例，但整体不构成完整的应用。

## 维护说明

- 本仓库主要用于长期保存和公开访问这些页面。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html`（如有）中的链接。
- 不建议对已有页面进行频繁修改，以保持归档的稳定性。
- 若页面内容已过时或不再需要，可将其移至 `archive/` 目录或直接删除。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可查看页面内容。
3. 也可通过 GitHub Pages 等服务直接部署本仓库，以在线浏览所有页面。
